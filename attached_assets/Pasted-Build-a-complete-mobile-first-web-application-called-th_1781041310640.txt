════════════════════════════════════════════════════════════

Jobber's UI patterns to replicate:
- White page backgrounds, light gray (#F9FAFB) section BG
- Clean sans-serif typography (Inter or system-ui)
- Simple top navbar with page title left, actions right
- Left sidebar navigation on desktop (collapsible)
- Card-based content with subtle drop shadows
- Status dots: green = active/approved, red = late/overdue,
  yellow = pending/requires action, gray = inactive
- Simple data tables with alternating row shading
- Large, clear action buttons with obvious labels
